Balise
Une balise est un élement d'une page.
Elle est constituée d'une balise ouvrante, d'une balise fermante et d'un contenu.
Exemple : 
<balise>Contenu</balise>

Une balise peut également ne pas avoir de contenu. Dans ce cas la, on peut la fermer directement.
Exemple : 
<br />
La balise <br /> sert à sauter une ligne.

Une balise peut également avoir des attributs. Les plus connus sont class et id. Ils sont notamment utilisés en CSS et en JavaScript
Exemple : 

<balise class="class" id="id">

	Contenu

</balise>

Sélecteurs
Un sélecteur est une chaîne de caractères permettant de sélectionner un ou plusieurs élements.

Les principaux sélecteurs sont les suivants :
Sélecteur de balise
Le sélecteur de balise est constitué uniquement du nom de la balise.
p va sélectionner toutes les balise <p></p> de la page.
Sélecteur par id
Le sélecteur par id est constitué d'un dièse (#) et du nom de l'id cible.
#target va sélectionner toutes les balise de la page ayant l'id target.
Sélecteur par classe
Le sélecteur par classe est constitué d'un point (.) et du nom de la classe cible.
.error va sélectionner toutes les balise de la page ayant la class error.

On utilise notamment les sélecteurs en CSS pour appliquer des styles aux élements, ou encore en JavaScript.
CSS
Langage permettant de définir le style d'une page web.
JavaScript
Langage permettant de manipuler dynamiquement une page web.
Commentaires
Les commentaires permettent de donner des informations sur une partie du code.
Ils ne sont pas interpétés par le navigateur.

Ils s'écrivent de différentes manière selon le langage :
Commentaire HTML : 
<!-- Mon commentaire ici -->
Commentaire CSS : 
/* Mon commentaire ici */
Commentaire JavaScript : 
// Mon commentaire sur une ligne
ou : 

/* Mon
commentaire
sur
plusieurs
ligne
*/
