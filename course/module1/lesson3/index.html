Introduction au CSS


Le CSS qu'est ce que c'est ?

CSS (ou Cascading Style Sheets) est un langage qui permet de mettre en forme le contenu HTML (via des couleurs, des polices...).

A quoi ça ressemble ?

Télecharger ce fichier et ouvrez le dans votre navigateur

Télecharger ce fichier et ouvrez le dans votre navigateur

Rajoutez la ligne de commande suivante dans la balise <head>:

<link rel="stylesheet" type="text/css" href="style.css">

Sans vous en rendre compte, vous venez d'appliquer une feuille de style à votre feuille HTML !

Comment ça marche ?

Schématiquement, une feuille de style CSS ressemble à cela :

sélecteur {
			    propriete1: valeur1;
			    propriete2: valeur2;
			    propriete3: valeur3;
			}

Une feuille de CSS contient donc:

des sélecteurs
qui permettront de sélectionner les balises ou des groupes de balises dont on veut modifier l'apparence
des propriétés
qui seront les différents « effets de style ». Par exemple, la propriété color qui permet d'indiquer la couleur du texte, font-size permet d'indiquer la taille du texte, etc.
une valeur
,qui est la valeur associé à chaque propriété. Par exemple, pour la propriété color, il faut indiquer le nom de la couleur. Pour font-size, il faut indiquer quelle taille on veut, etc.

Voici un petit exemple pour illustrer ceci:

p {
			    color: blue;         
			    font-family: Arial     
			    font-size:10px       
			}

Ceci indique que les paragraphes (correspondant à la balise <p>) seront tous bleu (color:blue), en police Arial (font-family: Arial), et d'une taille de 10 pixels (font-size:10px).


Comment lier CSS et HTML ?

Il existe plusieurs méthodes pour intégrer du CSS dans un fichier HTML

Ecrire dans la balise HTML

Il est possible d'écrire le code directement dans la balise HTML en utilisant l'attibut style

 <p style="color: blue;">Belleville Coding Club>

Cela est toutefois peu recommandé car complique beaucoup la relecture du code

Ecrire dans une balise <style>

Il est aussi possible d'écrire le code dans une balise &lgstyle>, elle même posée dans le <head> du fichier HTML

<!DOCTYPE HTML>
				<html>
					<head>
						<title>Ma page</title>
						<style>
						p
						{
							color: blues;
						}
						</style>
					</head>
					<body>
						<h1>Titre de page</h1>
						<h2>Titre</h2>
					</body>
				</html>

Ceci n'est à utiliser que si le code CSS de la page est unique.

Ecrire dans un fichier .css

Si l'on veut appliquer le même style à différentes pages d'un site web, on préferera écrire le code dans un fichier.css.

Dans chaque page HTML, il faudra utiliser la syntaxe suivante, et la placer dans le <head>

<link rel="stylesheet" type="text/css" href="style.css">

Voici un aperçu dans un squelette de page HTML: 

<!DOCTYPE HTML>
				<html>
					<head>
						<title>Ma page</title>
						<link rel="stylesheet" type="text/css" href="style.css">
					</head>
					<body>
						<h1>Titre de page</h1>
						<h2>Titre</h2>
					</body>
				</html>



Les différents sélecteurs

Un sélecteur permet de définir à quelles balises un style donné doit être appliqué. Il en existe plusieurs et permettent de sélectionner de différentes façons.

Les sélecteurs les plus courants:

ATTENTION : Ces sélecteurs s'appliquent à TOUTES les balises, pour appliquer un style à une balise, ou un groupe de balises particulière, regardez l'exemple "class et Id"

Type Balise

On l'a vu plus tôt, ce sélecteur permet d'appliquer un style à toutes les balises.

Voici un exemple sur les balises <p>: 

p {
				    color: blue;         
				    font-family: Arial  
				}

Type A et B

On utilise "A , B" pour sélectionner plusieurs balises

Voici un exemple sur les balises <h1> et <p>:

Au lieu d'écrire

h1 {
				    color: blue;         
				    font-family: Arial         
				}
				p {
				    color: blue;         
				    font-family: Arial         
				}

Nous pouvons simplifier en:

h1 , p {
				    color: blue;         
				    font-family: Arial         
				}

L'ensemble des titres <h1> et des paragraphes <p> apparaitront en bleu et en Arial.


Les sélecteurs avancés:

Type class et id

Seuls les sélecteurs "class et id" permettent d'appliquer un style à une balise (ou un groupe de balises) particulière

Les sélecteurs "class et id" sont très similaires, à une différence près :

Class
est réutilisable (applicable à différentes balises ou groupes de balises)

Voici un exemple sur les balises <p> et <em>:

Code HTML

< p class= "introduction" > La <em> plateforme d'expression libre </em> Belleville Blog </p>

Code CSS 

.introduction {
											    color: blue;         
											    font-family: Arial
											}
Id
n'est utilisable que sur une seule balise, donc une seule fois dans le code ! Son utilité très particulière sera vue plus tard, notamment dans le cours sur le javascript.

Voici un exemple sur les balises <p> et <em>:

Code HTML

< p id= "identifiant" > La <em> plateforme d'expression libre </em> Belleville Blog </	 >

Code CSS 

#identifiant {
											color: blue;         
											font-family: Arial
										}
a

Type A B

On utilise "A B" pour sélectionner toutes les balises A situées à l'intérieur d'une balise B.

Voici un exemple sur les balises <p> et <em>:

Code HTML

<p> La <em> plateforme d'expression libre </em> Belleville Blog </p>

Code CSS 

 p em {
				    color: blue;         
				    font-family: Arial
				}

L'ensemble des balises <em> contenu dans une balise <p> apparaitront en bleu et en Arial.

Type A + B

On utilise le sélecteur "A + B" pour sélectionner une balise B lorsqu'elle est située juste après une balise A

Voici un exemple sur les balises <h1> et <p>:

Code HTML

<h1>Belleville Blog</h1>
				<p>Belleville Blog</p>

Code CSS 

h1 + p {
				    color: blue;         
				    font-family: Arial      
				}

La première balise <p> située après toutes les balises <h1> apparaitront en bleu et en Arial.

Type A[attribut]

On utilise le sélecteur "A[xxx]" pour sélectionner une balise A possédant un l'attribut xxx

Voici un exemple:

Code HTML

<a href="http://bellevilleBlog.com" title="blank">

Code CSS 

a[blank] {
				    color: blue;         
				    font-family: Arial  
				}
