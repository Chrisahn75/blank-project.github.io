Leçon 2

Les bases d'HTML

Dans cette leçon vous apprendrez les bases d'HTML, le langage qui permet de définir la stucture d'une page web. Tous les sites web du monde sont écrit en HTML.
Vous ne le savez (peut-être) pas encore, mais cette page est écrite en HTML !
Pour vous en rendre compte, vous pouvez faire Clic-droit > Afficher le code-source de la page dans votre navigateur.
Vous voyez alors apparaitre tout le code soutce de la page, c'est à dire le HTML ainsi que d'autre élements que nous découvrirons plus tard.
