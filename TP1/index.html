HTML

Structure d'une page HTML

Une page HTML a la structure suivante :

<!DOCTYPE html>
<html>
     <head>
          <title>Le titre de la page ici</title>
     </head>
     <body>

     </body>
</html>
Description des balises utilisées :
<!DOCTYPE html>
ce n'est pas une balise standard, cette balise sert à indiquer que l'on utilise HTML 5
<html>
C'est la balise racine présente dans chaque page web.
<head>
C'est la balise qui contient les méta-données de la page
<title>
C'est la balise qui contient titre de la page, affiché en haut du navigateur.
<body>
C'est la balise qui contient le contenu de la page web, c'est-à-dire ce qui est affiché à l'écran

Présentation des principaux élements HTML

p

La balise p sert à définir un paragraphe. Elle est bordée par des espaces, et peut contenir du texte et d'autres élements

a

La balise a sert à définir un lien.
Exemple :
<a href="http://www.google.com" >Un lien vers Google</a>
Le code ci-dessus donne le résultat suivant : Un lien vers Google

img

La balise img sert à définir une image.
Exemple :
<img src="img/sample_image.png" alt="Une image d'exemple" />
